# python

## HTTP server

Python 2

```
python -m SimpleHTTPServer 8000
```

Python 3

```
python3 -m http.server 8000
```

## Virtualenv

```
sudo apt install -y python3-venv
python3 -m venv venv
source venv/bin/activate
```

## pip

```
pip install --upgrade --force-reinstall -r requirements.txt 
```
